# Ruby-Game-of-Life

Ruby implementation of the Conway's Game of Life.


# Usage

run lifegame
```ruby
$ rake run
 or
$ ruby lib/main.ruby
```

If you want to install a dependency library
```ruby
$ bundle install
```

# Key command

*   Start: `x` or `X`
*   Pause: `p` or `P`
*   Next : `n` or `N` or `(space key)`
*   Reset: `r` or `R`
*   Quit : `q` or `Q`
